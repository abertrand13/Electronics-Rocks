Name flipped ( )
Description Returns a copy that is flipped 180 degrees from the vector's forward angle.
Syntax 
flipped();
Returns new vector that is flipped 180 degrees from the vector (Vect2)
Usage Web & Application
Related flip ( )
rotatedRight ( )
rotatedLeft ( )
Examples 
import point2line.*;

Vect2 velocity = new Vect2( 0, 1 );

println( degrees( velocity.angle() ) );  // prints "90.0"

Vect2 reverse = velocity.flipped();

println( degrees( reverse.angle() ) );  // prints "270.0"
