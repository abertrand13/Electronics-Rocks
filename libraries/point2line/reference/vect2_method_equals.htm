Name equals ( )
Description Returns a boolean that is true if the vector is equals an input vector.
Syntax 
equals(vector);
Parameters
vector   to compare against (Vect2)
Returns true if the given vector is equal this vector (boolean)
Usage Web & Application
Related
Examples 
import point2line.*;

Vect2 velocity1 = new Vect2( 1, -1 );
Vect2 velocity2 = new Vect2( 1, 1 );

println( velocity1.equals( velocity2 ) ); // prints "false"
println( velocity2.equals( velocity1 ) ); // prints "false"

velocity1.y = 1;

println( velocity1.equals( velocity2 ) ); // prints "true"
println( velocity2.equals( velocity1 ) ); // prints "true"
