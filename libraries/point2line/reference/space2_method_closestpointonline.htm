Name closestPointOnLine ( )
Description Calculates the closest point on a continuous line from a given point using (orthogonal) projection of a vector onto another vector.
Syntax 
closestPointOnLine(point, linePointA, linePointB);
Parameters
point   (Vect2)
linePointA   (Vect2)
linePointB   (Vect2)
Returns closest point on the line (Vect2)
Usage Web & Application
Related
Examples 
import point2line.*;

Vect2 linePoint1 = new Vect2( 0, 0 );
Vect2 linePoint2 = new Vect2( 10, 10 );
Vect2 testPoint = new Vect2( 100, 0 );

Vect2 closestPoint = Space2.closestPointOnLine( testPoint, linePoint1, linePoint2 );

println( closestPoint ); // prints Vect2( 50.0, 50.0 )
